Introduction:

JOS is invented in JDollar by wilmix jemin j at 2015 and is used only for the construction of Operating Systems Using JOS One can create an OS with in 5 or 10 or 30 minutes. JOS takes only .OS Extension file and converts to byte code which can directly interact with C@ , and JDollar class file. So that we can call JDollar class file or C@ class file in JOS Program.
